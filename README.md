# Database Integration System

## Overview

This project was developed as part of Week 3 of the Full Stack Development Industrial Training Program. The objective of the project is to integrate a backend application with a database to enable persistent data storage and management. The system allows users to perform CRUD (Create, Read, Update, Delete) operations on student records through a Flask-based backend connected to an SQLite database.

The project demonstrates the fundamentals of database design, backend integration, and data management, providing a practical understanding of how modern web applications store and retrieve information efficiently.

## Features

* Database integration using SQLite
* Create new student records
* Retrieve stored student records
* Update existing student information
* Delete student records
* RESTful API implementation
* Input validation and proper data handling

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
* JavaScript

## Project Structure

* **app.py** – Flask backend application
* **students.db** – SQLite database file
* **templates/index.html** – User interface for data entry and interaction

## CRUD Operations Implemented

### Create

Adds a new student record to the database.

### Read

Retrieves and displays all stored student records.

### Update

Modifies existing student information.

### Delete

Removes student records from the database.
